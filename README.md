## Getting Started

```bash
# Clone the repository
git clone https://github.com/maillab/cloud-mail-docs.git

# Go into the folder
cd cloud-mail-docs

# Install dependencies
pnpm install

# Build docs
pnpm run docs:build

# Run locally
pnpm run docs:dev
````
